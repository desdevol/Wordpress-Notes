## Set Equal Height with Elementor

[Source: GeneratePress](https://generatepress.com/forums/topic/elementor-column-equal-height/)

Add this class to custom CSS and apply it on desired item.

```css
.equal-height-content {
    height: 100%;
    display: flex;
}
```
